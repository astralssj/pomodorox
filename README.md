# Pomodoro App

A simple desktop Pomodoro timer built with Electron, HTML, and Tailwind CSS.

## Features
- 25-minute timer with start button
- Real-time date and time display
- Notepad for logging activity during each session
- Circular progress clock with count of completed sessions
- Left sidebar to-do list with delete capability and a history log on the right
- Dark, modern UI with profile themes (Aarhon or Paula)

## Getting Started
1. Install dependencies: `npm install`
2. Run: `npm start`

## Development
- Add build steps to `package.json` as needed.
